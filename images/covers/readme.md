# /images/covers

Optional. Drop per-episode cover art here (square, at least 400×400px)
and set the `cover` field in `data/episodes.js`, e.g.
`images/covers/ep-01.jpg`.

Leave `cover: ""` and the episode card shows the gradient episode
number instead of artwork — that's the default and looks intentional
on its own, so don't feel obligated to fill this in for launch.
